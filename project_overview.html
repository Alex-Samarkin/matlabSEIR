Обнаружение волн Алгоритмы

Назначение: Алгоритмическое обнаружение волн заражения через поиск локальных минимумов на сглаженной кривой случаев. Присвоение номеров волн каждому дню.

  • rolling_mean() — скользящее среднее (окно 21 день)
  • find_wave_minima() — поиск минимумов с проверкой prominence
  • assign_waves() — присвоение номеров волн
  • plot_waves_gr() — визуализация волн с зонами
  • select_wave() — фильтрация по стране и номеру волны
  • plot_wave_detail() — детальный график волны с вариантами
  • Минимальное расстояние между волнами: 31 день
  • Порог prominence: 18% от максимального пика
function find_wave_minima(smoothed; min_prominence = 0.15, min_distance = 30) n = length(smoothed) global_max = maximum(smoothed) threshold = global_max * min_prominence minima = Int[] # Поиск локальных минимумов с проверкой prominence for i in 2:(n-1) if smoothed[i] < smoothed[i-1] && smoothed[i] < smoothed[i+1] if smoothed[i] < threshold # Проверка минимального расстояния if isempty(minima) || (i - last(minima)) >= min_distance push!(minima, i) end end end end return minima end

Полный пайплайн Калибровка Статистика

Назначение: Полный цикл моделирования SEIRD: классическая модель (RK4) vs дробная модель Капуто. Калибровка параметров, статистическое обоснование α, анализ остатков.

  • Классическая SEIRD — решение методом RK4
  • Дробная SEIRD — производная Капуто, метод ABM (Adams-Bashforth-Moulton)
  • Калибровка — оптимизация β (и α для дробной) через L-BFGS
  • AIC/BIC/AICc — информационные критерии для выбора модели
  • LRT тест — тест отношения правдоподобия
  • Профиль правдоподобия α — 95% доверительный интервал
  • Анализ остатков — автокорреляция, тест Дарбина-Уотсона, RMSE, MAE
  • 6-панельная визуализация — все результаты в одном figure
# Дробная SEIRD — производная Капуто function solve_fractional(α, β, σ, γ, μ, N, I0, E0, n_days) h = 1.0 # шаг = 1 день ns = n_days y0 = [N - I0 - E0, E0, I0, 0.0, 0.0] Y = zeros(ns + 1, 5) Γα1 = Γ(α + 1) Γα2 = Γ(α + 2) for n in 1:ns # Предиктор (явная схема) pred = copy(y0) for j in 0:(n-1) a_j = ((n - j)^α - (n - 1 - j)^α) / Γα1 pred .+= h^α * a_j .* fhist[j + 1, :] end # Корректор (неявная схема) fp = f(pred) corr = copy(y0) # ... суммирование по истории Y[n + 1, :] .= corr end return Y end

🧮 Математические модели

Классическая SEIRD модель

dS/dt = -β·S·I/N # Восприимчивые → Экспонированные dE/dt = β·S·I/N - σ·E # Экспонированные → Инфицированные dI/dt = σ·E - γ·I - μ·I # Инфицированные → Выздоровевшие / Умершие dR/dt = γ·I # Выздоровевшие dD/dt = μ·I # Умершие # Параметры: β = скорость заражения (калибруется) σ = 1 / инкубационный_период γ = (1 - CFR) / инфекционный_период μ = CFR / инфекционный_период

Дробная модель Капуто

D^α_C S = -β·S·I/N D^α_C E = β·S·I/N - σ·E D^α_C I = σ·E - γ·I - μ·I D^α_C R = γ·I D^α_C D = μ·I # α ∈ (0, 1] — порядок дробной производной # При α = 1: классическая SEIRD # При α < 1: "память" системы (эффект запаздывания) # Метод решения: Adams-Bashforth-Moulton # Сложность: O(n²) из-за ядра памяти

Интерпретация α

Диапазон α Интерпретация Рекомендация
> 0.95Почти марковская динамикаКлассическая SEIRD достаточна
0.80–0.95Умеренная памятьДробная модель предпочтительна
0.65–0.80Сильная памятьГетерогенный популяционный отклик
< 0.65Экстремальная памятьРассмотреть пространственные модели

Статистические критерии

  • AIC (Akaike Information Criterion): 2k + 2·NLL — штраф за сложность модели
  • BIC (Bayesian Information Criterion): k·log(n) + 2·NLL — более строгий штраф
  • AICc: скорректированный AIC для малых выборок
  • LRT (Likelihood Ratio Test): Λ = 2(NLL₀ - NLL₁) ~ χ²(Δk)
  • Профиль правдоподобия: 95% ДИ для α через χ²₁(0.95)
  • Тест Дарбина-Уотсона: DW = Σ(Δres)² / Σ(res)² — автокорреляция остатков
