## Ключевые возможности

- **Загрузка данных WHO** — автоматическая загрузка ежедневной статистики COVID-19
- **Обнаружение волн** — алгоритмическое выявление волн заражения через скользящие средние
- **Сопоставление вариантов** — привязка периодов доминирования вирусных вариантов
- **SEIRD моделирование** — классическая (RK4) и дробная (Caputo) модели
- **Статистическая валидация** — AIC/BIC, тесты правдоподобия, профиль доверия α
- **Визуализация** — графики уровня Nature/Science с colorblind-safe палитрой

## 📜 Скрипты Julia

### load_data.jl

**Назначение:** Загрузка ежедневных данных COVID-19 из официального источника WHO, предобработка (интерполяция пропусков, сглаживание), создание агрегированных датафреймов по странам.

**Особенности:**
- Автоматическая загрузка с WHO CDN (кэширование в `covid_all_daily.csv`)
- Линейная интерполяция пропущенных значений
- 7-дневное скользящее среднее для сглаживания
- Расчёт ежедневного прироста случаев
- Настройки визуализации по стандарту Nature/Science

---

### load_data2.jl

**Назначение:** Улучшенная версия загрузчика с автоопределением среды выполнения, кроссплатформенной поддержкой GUI и расширенной обработкой ошибок.

**Особенности:**
- Автоопределение бэкенда (REPL, Jupyter, скрипт)
- Функция `show_plot()` для безопасного отображения графиков
- Расширенная интерполяция с обработкой граничных случаев
- Универсальная функция `fill_daily_grid()`

---

### waves_detection.jl

**Назначение:** Алгоритмическое обнаружение волн заражения через поиск локальных минимумов на сглаженной кривой случаев. Присвоение номеров волн каждому дню.

**Особенности:**
- `rolling_mean()` — скользящее среднее (окно 21 день)
- `find_wave_minima()` — поиск минимумов с проверкой prominence
- `assign_waves()` — присвоение номеров волн
- Минимальное расстояние между волнами: 31 день
- Порог prominence: 18% от максимального пика

---

### waves_and_stamms.jl

**Назначение:** Создание таблицы параметров вариантов вируса, визуализация временных линий доминирования штаммов, радарные диаграммы для сравнения характеристик.

**Особенности:**
- Таблица биологических параметров 8 вариантов (Wuhan → JN.1)
- Расчёт продолжительности доминирования
- Функция `values_for_date()` — определение активного варианта на дату
- Интерполяция параметров при перекрытии периодов
- Нормированные радарные диаграммы (min-max scaling)

---

### seir modelling.jl

**Назначение:** Детальная визуализация отдельной волны заражения с наложением зон доминирования вариантов и графика R₀.

**Особенности:**
- Загрузка данных из `cases_with_waves.csv`
- Функция `select_wave()` — выбор конкретной волны страны
- `plot_wave_detail()` — график с зонами штаммов
- Двойная Y-ось: случаи + R₀
- Цветовое кодирование вариантов

---

### seird_pipeline.jl

**Назначение:** Полный цикл моделирования SEIRD: классическая модель (RK4) vs дробная модель Капуто.

**Особенности:**
- **Классическая SEIRD** — решение методом RK4
- **Дробная SEIRD** — производная Капуто, метод ABM
- **Калибровка** — оптимизация β (и α для дробной) через L-BFGS
- **AIC/BIC/AICc** — информационные критерии для выбора модели
- **6-панельная визуализация** — все результаты в одном figure

## 🧮 Математические модели

### Классическая SEIRD модель

```
dS/dt = -β·S·I/N          # Восприимчивые → Экспонированные
dE/dt = β·S·I/N - σ·E     # Экспонированные → Инфицированные
dI/dt = σ·E - γ·I - μ·I   # Инфицированные → Выздоровевшие / Умершие
dR/dt = γ·I               # Выздоровевшие
dD/dt = μ·I               # Умершие
```

**Параметры:**
- β = скорость заражения (калибруется)
- σ = 1 / инкубационный_период
- γ = (1 - CFR) / инфекционный_период
- μ = CFR / инфекционный_период

---

### Дробная модель Капуто

```
D^α_C S = -β·S·I/N
D^α_C E = β·S·I/N - σ·E
D^α_C I = σ·E - γ·I - μ·I
D^α_C R = γ·I
D^α_C D = μ·I
```

- **α ∈ (0, 1]** — порядок дробной производной
- При α = 1: классическая SEIRD
- При α < 1: "память" системы (эффект запаздывания)
- Метод решения: Adams-Bashforth-Moulton

---

## Интерпретация α

| Диапазон α | Интерпретация | Рекомендация |
|------------|---------------|--------------|
| > 0.95 | Почти марковская динамика | Классическая SEIRD достаточна |
| 0.80–0.95 | Умеренная память | Дробная модель предпочтительна |
| 0.65–0.80 | Сильная память | Гетерогенный популяционный отклик |
| < 0.65 | Экстремальная память | Рассмотреть пространственные модели |

---

## Статистические критерии

- **AIC (Akaike Information Criterion):** 2k + 2·NLL — штраф за сложность модели
- **BIC (Bayesian Information Criterion):** k·log(n) + 2·NLL — более строгий штраф
- **AICc:** скорректированный AIC для малых выборок
- **LRT (Likelihood Ratio Test):** Λ = 2(NLL₀ - NLL₁) ~ χ²(Δk)
- **Профиль правдоподобия:** 95% ДИ для α через χ²₁(0.95)
- **Тест Дарбина-Уотсона:** DW = Σ(Δres)² / Σ(res)² — автокорреляция остатков
